Kommissionen beslutar inleda ett överträdelseförfarande mot Tyskland på grund av Tysklands vägran att häva embargot för brittiskt nötkött

Europeiska kommissionen har i dag beslutat att inleda ett formellt rättsligt förfarande mot Tyskland för dess vägran att upphäva embargot för brittiskt nötkött. Kommissionen anser att Tyskland inte har uppfyllt sina skyldigheter enligt rådets beslut 98/256/EG och kommissionens beslut 1999/514/EG om att häva embargot. Som ett första steg i överträdelseförfarandet kommer David Byrne, EU:s kommissionär för hälsa och konsumentskydd, att be den tyska regeringen inkomma med sina synpunkter så snart som möjligt efter den 17 mars. Om kommissionen inte fått ett tillfredsställande svar då, kan den besluta att skicka ett så kallat motiverat yttrande till Tyskland med begäran att Tysklands anförda överträdelse måste upphöra.

Kommissionen hoppas emellertid att saken inte ska behöva dras inför domstol. Efter kommissionens sammanträde i Strasbourg, sade David Byrne: "Jag har haft konstruktiva kontakter med den tyska regeringen och det finns positiva tecken. Jag måste dock se till att EU-lagstiftningen följs, särskilt eftersom Tyskland dröjt så länge med att häva förbudet för brittiskt nötkött. Vi kommer emellertid utan tvivel att kunna undvika att saken dras inför domstol om Tyskland snabbt börjar att följa lagen."
